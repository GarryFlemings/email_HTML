This signature block is at the bottom of your developer's emails.
  1. A blank line to separate from text above.
  2. Three dots to remind him to use a closing.
  3. A blank line.
  1. His name.
  2. A picture and traditional contact information.
  3. Four lines to separate from the next message in a chain, should he use this for replies.
Do you want a more elaborate signature block? You have access to all the same funtionality mentioned in the Mini-Blog  section.

What can you notice about this file?
  • The Mini-Blog  section can be an inspiration to you for having us develop periodic newsletters to your contacts.
  • The Signature Block  section can be an inspiration to you for a particularly attractive and useful ending to all your emails. Some companies get extra branding impact by having all sales and marketing employees (or all employees) use the same signature block (personalized for each employee).
  • The colors, fonts, sizes, and formats of web sites are available to you.
  • The contrast between letters and backgrounds (the difference in light and dark) is enough for most visually-impaired contacts to see. If you want to know details on the web standard, let us know. If you prefer to depend on us for this, we're ready!
  • No part of this file includes light text on a dark background. A small number of email programs still don't show background colors. For those, the result is light text on a white background. Maybe ... not for you!
  • The file is responsive—it adapts the presentation to the size of the reader's window. Try it! Open the file in a browser and move the left and right sides closer together, then farther apart.
  • You can use a divider  (like the pink-red horizontal line above this box) to create a visual border between parts of your letter. You can choose the color and width.
  • You can arrange text in any number of columns you like (like newspapers). This file uses three columns just below the Mini-Blog  heading. It uses two columns just below that and ... well, here.
  • For narrow screen sizes, all columns fold gracefully under one another. Check that, too!
  • Accompanying this file is a plain text version. Few SPAM authors include this, so SPAM filters are less likely to block your email. Further, some contacts may configure their email programs for plain text, if available. You'll reach them more gracefully.
