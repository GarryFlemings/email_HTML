Email Mini-Blog
------------------------------

Why a text-only version?
* Some email programs don't support any HTML (the presentation language of the web). This helps you reach your contacts using those programs.
* Some email programs allow readers to opt for plain text, if available; some people prefer the option. Reach them gracefully, too!
* Spam filters are less likely to block your email. Few spammers include a text-only version. You set yourself apart from them.

ABOUT THE HTML VERSION OF THIS EMAIL

When this file is properly sent as email, it looks great almost anywhere it goes.
* Wide screens and narrow screens.
* Phones, tablets, computers, and more.
* "Apple Macintosh", "Microsoft Windows", "Apple iOS", "Android", and other operating systems.
* Email applications native to operating systems ("Apple Mail"), or available as add-ons ("Microsoft Outlook"); mail viewers inside various browsers, etc.
* Email providers ("Gmail", "Outlook.com", Internet Service Providers (ISPs), corporate email systems, etc.)
* Old versions of all the above.

You can test different widths pretty easily in browsers and on different devices. Properly sending the HTML file as email is more difficult. Let us know; we're happy to help you at no cost!

The value to you: You can be confident that when you send a beautiful email to a list of your contacts, every contact will see a message that best represents you.

What can you notice about the HTML file?
--------------------------------------------------------------

* The "Mini-Blog" section can be an inspiration to you for having us develop periodic newsletters to your contacts.
* The "Signature Block" section can be an inspiration to you for a particularly attractive and useful ending to all your emails. Some companies get extra branding impact by having all sales and marketing employees (or all employees) use the same signature block (personalized for each employee).
* This plain text file accompanies the HTML. Few SPAM authors include this, so SPAM filters are less likely to block your email. Further, some contacts may configure their email programs for plain text, if available. You'll reach them more gracefully.
* Mostly in the HTML version:
 ** The colors, fonts, sizes, and formats of web sites are available to you.
 ** The file is "responsive"--it adapts the presentation to the size of the reader's window.
 ** You can arrange text in any number of columns you like (like newspapers). Parts of the HTML file use three columns; parts use two.
 ** For narrow screen sizes, all columns are gracefully under one another.
 ** You can use a divider (in this file, we use lines of many equal signs) to create a visual border between parts of your letter. In HTML, you can choose the color and size.
 ** The contrast between letters and backgrounds (the difference in light and dark) is enough for most visually-impaired contacts to see. If you want to know details on the web standard, let us know. If you prefer to depend on us for this, we're ready!
 ** No part of this file includes light text on a dark background. A small number of email programs still don't show background colors. For those, the result is light text on a white background. Maybe ... not good enough for you!

* Areas where the reader can choose to view one of several pieces of content at a time (often called an accordian)
